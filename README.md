# singlechip51-alarm-clock
## 基本功能
基于51单片机的多功能时钟
* 使用数码管显示，采用锁存器进行动态刷新，定时0计数。
* 使用键盘扫描，静态变量捕获键盘值。
* 使用蜂鸣器，闹钟功能和整点报时。
